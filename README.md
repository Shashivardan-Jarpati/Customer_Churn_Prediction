# Customer Churn Prediction

This project predicts whether a customer will churn (leave) or stay
using machine learning classification techniques.

## Dataset
- Telco Customer Churn Dataset
- Target column: Churn (Yes / No)

## Data Preprocessing
- Handled missing values
- Encoded categorical features
- Converted target labels to binary format

## Algorithm Used
- Logistic Regression

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Train the model:
   python train.py
3. Predict customer churn:
   python predict.py

## Output
- 0 → Customer stays
- 1 → Customer churns
